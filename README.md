# 🎬 YouTube Clone (JavaScript)

A fully interactive YouTube homepage clone built using **HTML5**, **CSS3**, and **Vanilla JavaScript**.

This repository continues from my HTML/CSS version by adding real user interactions, dynamic rendering, and modern JavaScript concepts.
